Tentang aplikasi

Perhitungan debit dilakukan berdasarkan persamaan sebagai berikut:
Q = CdBa√2gh1
Koefisien debit pada kondisi aliran bebas (free flow) adalah:
Cd = cc √1+η
Koefisien debit pada kondisi aliran tenggelam (submerged flow) adalah:
Cd = Cc [ √ξ − √ξ2 − (1/(ηη) − 1)2(1 − 1/λ2) ]1/2 1/η − η
Aliran tenggelam terjadi jika:
h1 > h2 ≥ Cca 2 (√(1 + 16 η(1+η) ) − 1)
Keterangan:
Cd = koefisien debit
a = bukaan pintu (m)
B = lebar pintu (m)
g = percepatan gravitasi (9,81 m/s2)
h1 = tinggi muka air di hulu pintu (m)
h2 = tinggi muka air di hilir pintu (m)
Q = debit (m3/s)
Cc = koefisien kontraksi
λ, η dan ξ merupakan parameter dihitung menggunakan persamaan berikut:
λ = h1/h2 η = Cca/h1 ξ = (1/η − 1)2 + 2(λ − 1)
